# Automation Governance Model

## Leadership Responsibilities

• Define automation roadmap and coverage targets  
• Align automation execution with release schedules  
• Supervise automation engineers and code reviews  
• Monitor execution stability and script health  

## Team Collaboration

• Manual testers identify automation candidates  
• Automation engineers develop and maintain scripts  
• QA Lead reviews execution metrics and quality risks  

## Governance Practices

• Sprint automation planning sessions  
• Script review checkpoints  
• Automation execution reporting  
• Continuous improvement discussions
